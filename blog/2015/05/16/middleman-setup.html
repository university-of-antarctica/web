  1. Should I use Middleman?

This post won't have a detailed discussion of the virtues of different static site generators. There are <%= link_to 'Too many generators' 'https://staticsitegenerators.net/' %> to do a proper comparison (394 at the time of writing).

After setting up Middleman, our development cycle consists of the following steps:

a) >> middleman server
b) Use a text editor to modify files in the Middleman 'source' directory 
c) Navigate a browser to localhost:4567 to look at the site
d) repeat b) and c) until you are satisfied with your changes
e) commit to git
f) >> middleman deploy
