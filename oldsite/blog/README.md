To install jekyll you need:
Ruby,
  sudo apt-get install ruby-full
  sudy yum install ruby
RubyGems
  download rubygems from https://rubygems.org/pages/download
  unzip package
  cd into folder
  run sudo ruby setup.rb
Linux
NodeJS
  install npm and node....


then run
  gem install jekyll
